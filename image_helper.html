📋 使用说明

如何获取图片：

  1. 点击下方的搜索链接，会在新标签页打开搜索结果
  2. 选择合适的图片，右键点击"另存为"
  3. 保存到项目的 images/ 文件夹
  4. 使用指定的文件名保存
  5. 刷新网站页面查看效果

🔧 批量下载脚本

如果您熟悉编程，可以使用以下Python脚本批量下载图片：

# 安装必要的库
pip install requests pillow beautifulsoup4

# 使用脚本下载图片
python download_naruto_images.py

脚本使用说明：

  1. 确保您有Python环境
  2. 安装必要的库
  3. 运行脚本，会自动搜索并下载图片
  4. 脚本会尝试获取免费的图片资源

📱 移动端操作指南

在手机上获取图片：

  1. 点击搜索链接
  2. 在图片搜索结果中，长按图片
  3. 选择"保存图片"或"下载图片"
  4. 通过云盘或数据线传输到电脑
  5. 重命名并放到项目文件夹
